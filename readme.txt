/*  Gestion de la base de données MYSQL   */

Ci-joint dans le dossier "sql", vous trouverez les deux fichiers correspondant à la création des tables requises (exporter via PHPMYADMIN).
Veuillez soit créer une base de données se nommant "php_project" soit modifier fichier "db.php" ligne 8.
Dans le fichier "db.php" ligne 8, veuillez remplacer "root","" par vos propres identifiants PHPMYADMIN.
Utilisez les fichiers présent dans le dossier "sql" afin de remplir la base de données.


/*  Informations  */

Le couple de connexion de base est "admin/admin".
Vous trouverez dans le dossier "csv" un fichier csv servant d'exemple pour l'onglet "Import" du site.
